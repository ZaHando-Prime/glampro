## 0.1.0

- Initial release of the `glam_pro_chat` Flutter package.
- `GlamProChat` widget with configurable `apiUrl`, `appBarTitle`, and `inputHint`.
- Rose & gold Glam Pro design system (Poppins via google_fonts).
- Gradient app bar with live status indicator (connecting / online / typing / offline).
- Animated three-dot typing indicator.
- Multi-line chat input with animated send button.
- RTL-aware layout for Arabic support.
- Provider-based state management (`ChatProvider`).
- Graceful error and retry states.
- Welcome message on session start.
- Full example app in `example/`.
